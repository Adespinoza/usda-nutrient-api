### Local Development
To locally test this project run the following commands:
```bash
# Clone this repo
git clone git@github.com:Adespinoza/usda-nutrient-api.git

# cd into it
cd usda-nutrient-api

# Install dependencies
bundle install
cd client
npm install

# Run Rake Task
cd ..
rake start
```
